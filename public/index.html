Hello There

This is a simple website to check if the Washing Machine is free, so you won't need to walk around with your clothes or ask in the whatsapp group.

The usage is very simple:

  1. Enter your name at the end of this text and click on: "Ok Let's GO MFJC!!!"
  2. Allow the website to send you notifications
  3. If the washing machine is Free you will be able to set an alarm, so you can pick up your clothes in time.
  4. If the washing machine is Busy, you will be able to see who's using it and how much time is left.
  5. Once you've finished with your clothes don't forget to toggle the button to Free.

Ok, that's all !!!
